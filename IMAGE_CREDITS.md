# Créditos de imágenes adicionales ANTARESa

Las 20 imágenes nuevas del catálogo son assets demo integrados para mostrar una carta más completa. Algunas fueron generadas específicamente para este proyecto y otras provienen de Wikimedia Commons, luego recortadas y gradeadas para mantener una estética coherente con la marca ANTARESa.

> Nota profesional: antes de uso comercial definitivo, conviene reemplazar estas imágenes demo por fotografías propias del local o revisar individualmente las licencias de cada fuente.
